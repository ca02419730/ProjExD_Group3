# ラストコカー・コカー

## 実行環境の必要条件(多分)
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* ふたつゲームがあるため、遊び方は異なる
* 今回は2つのゲームを制作した。１つ目はランダムな数字の枠を通過することで装備を強化し、敵を攻撃して倒し続けるシューティングゲームである。
2つ目はランダムな数字の枠を通過することでこうかとんが増加させ、ステージBOSSと対戦するゲーム。

## ゲームの遊び方
* アローモードの場合： HPが0になるとゲームオーバーとなる。矢印キーでこうかとん上下に操作し、2つのうちの片方の枠を通過して、こうかとんの装備を強化させる。
* ミリタリーモードの場合：自軍(こうかとんの数)が0になるとゲームオーバーとなる。矢印キーでこうかとん左右に操作し，2つのうちの片方の枠を通過して、こうかとんの自軍を増減させる。

## ゲームの実装
### 共通基本機能
* 2つのゲームの選択画面であり、2つのモードから選択してもらう。
* 画面が自動に縦スクロールされる。
* キャラクターを上下もしくは左右のみにに移動することが出来る。
* ランダムに生成された2つの数字の枠組みを上から下へ流す。
* 生成された数字を潜ることにより数字が増減される。
* 2つのうち、片方の数字の枠組みのみ取得する。


### 分担追加機能
* ホーム画面機能（担当：羽地）：プレーヤーに任意のゲームを選択させるスタート画面に関するクラス
* コカーコカー（アローモード）もしくはラストコカー(ミリタリーモード)を選択した際に別の実行ファイルを呼び出す
* ミリタリーモード機能（担当：長澤、佐野）：こうかとんの自軍、乱数枠、BOSSとの戦闘に関するクラス
1. 2つまとめて取得したら、左の方を採用する
2. 1個取ったら、もう片方が消える
3. こうかとんの画像にする
4. 有利になるものを取ったら、笑顔になり、不利になるものを取ったら、悲しい表情になる

* アローモード機能（担当：鹿股、高以良）：こうかとんの弓、乱数枠、敵との戦闘に関するクラス
* コカーコカーにおいて、こうかとんが壁を通るとアイテムを取得できるようにする 
* コカーコカーにおいて、敵を攻撃できるようにする

* 広告機能（担当：巻）：無料スマホゲーム特有の広告を途中に挿入するクラス

### メモ
* ゲームが開始されるとProExD_group3_titleからshine.py(ミリタリーモード)、もしくはkokakoka.py(アローモード)に移行する
* ミリタリーモードでは自軍の数を増やして、軍事力を拡大するゲームである。
