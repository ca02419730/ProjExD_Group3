# 丸焼き豪華豚

## 実行環境の必要条件(多分)
* python >= 3.10
* pygame >= 2.1


## ゲームの概要
段々大きい数にしていくゲームである

## ゲームの遊び方
* 矢印キーでキャラクターを操作していくゲーム
* 0になったらゲームオーバーとなる


## ゲームの実装
### 共通基本機能
* 2つのモードがあって、選んでもらう

### 分担追加機能
* 私が担当する部分は、基本機能に加えて、以下の機能を追加する
1.2つまとめてとったら、左の方を採用する
2.1個取ったら、もう片方が消える
3.こうかとんに変更ができる
4.有利になるものを取ったら、笑顔になり、不利になるものを取ったら、悲しい表情になる
