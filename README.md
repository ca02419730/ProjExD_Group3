# ラストコカー・コカー

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 敵を攻撃して倒し続けるゲーム

## ゲームの遊び方
* こうかとんを縦横、または上下に動かして、敵を攻撃する。
* ふたつゲームがあるため、遊び方は異なる

## ゲームの実装
### 共通基本機能
* 2つのゲームの選択画面

### 分担追加機能
* コカーコカーを選択した際に別のpyファイルを開くようにする
* コカーコカーにおいて、こうかとんが壁を通るとアイテムを取得できるようにする 
* コカーコカーにおいて、敵を攻撃できるようにする
